Building a WinUI 3 desktop app (CanfarDesktop) that interfaces with the Canadian Astronomy Data Centre (CADC) TAP service. The app targets astronomers who understand ADQL but want visual query building for convenience. The user has a macOS reference app and is building Windows-specific features beyond what the macOS version offers. They are comfortable with the full stack: XAML, C#, MVVM, ADQL, and the CADC API.
