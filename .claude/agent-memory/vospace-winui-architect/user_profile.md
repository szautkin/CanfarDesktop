Serhii Zautkin is the sole developer of the Verbinal desktop clients for the CANFAR Science Portal:
- **Windows app**: CanfarDesktop (WinUI 3, .NET 8, C#) — this repo
- **macOS app**: canfar-macos/Verbinal (SwiftUI, Swift) — sibling repo

Both apps target astronomers who use CADC infrastructure for compute sessions (Skaha), archive search (TAP/ADQL), and cloud storage (VOSpace/ARC). The apps are published to their respective app stores (Microsoft Store v1.0.5.0 as of 2026-01).

Serhii works across both codebases and keeps feature parity in mind. He thinks in terms of the astronomer's end-to-end workflow: search archive, download data, analyze, store results.
